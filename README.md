# LuzzlePeak
2D puzzle game based on Nintendo 64's "Pokémon Puzzle League"

Blocks are destroyed by aligning them in a row of at least three equal-typed blocks.

Controls:

W	Move cursor Up
A	Move cursor Left
S	Move cursor Down
D	Move cursor Right
Space	Swap blocks denoted by cursor
